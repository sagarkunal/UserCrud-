A full-stack CRUD (Create, Read, Update, Delete) application built using the MERN stack with API integration via Axios. This project demonstrates end-to-end data flow from frontend to backend using RESTful APIs.

🛠️ Tech Stack
Frontend: React.js, Bootstrap / Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB (Mongoose)
API Handling: Axios
Version Control: Git & GitHub

📌 Features
✅ Create new users
✅ View all users
✅ Update user details
✅ Delete users
✅ RESTful API integration
✅ Responsive UI
✅ Toast notifications for actions

🔗 API Endpoints
Method	Endpoint	Description
GET	/api/users	Get all users
POST	/api/users	Create user
PUT	/api/users/:id	Update user
DELETE	/api/users/:id	Delete user
